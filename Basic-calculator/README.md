# Basic Python Calculator

This is my first calculator project built using Python and Tkinter.  
It was created to understand the basics of GUI programming and event handling.

## Features
- Basic arithmetic operations (+, -, ×, ÷)
- Button based input system
- Simple graphical interface using Tkinter
- Real-time result display

## Technologies Used
- Python
- Tkinter (Python GUI library)
