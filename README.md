# Python Calculator Projects

This repository contains multiple versions of a calculator built using Python and Tkinter.

The goal of this repository is to show the evolution of my GUI development skills.

---

## Projects Included

### 1. Basic Calculator
My first attempt at building a calculator using Python Tkinter.

Features:
- Basic arithmetic operations
- Simple GUI interface
- Button-based input

Location:
basic-calculator/

---

### 2. Advanced GUI Calculator
An improved version with better UI design and more features.

Features:
- Dark theme interface
- Backspace button
- Keyboard input support
- Improved code structure

Location:
advanced-calculator/

---

## Technologies Used
- Python
- Tkinter
